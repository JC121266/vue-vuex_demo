# vue-vuex_demo
``` bash
# first
git clone https://github.com/JC121266/vue-vuex_demo.git

# second
cd vue-vuex_demo

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# sftp server
npm run build:sftp
```
vue, vuex, vue-router, vux, axios, gulp, gulp-sftp
